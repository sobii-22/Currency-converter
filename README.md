# Currency Converter

A simple and interactive currency converter web application built with HTML, CSS, and JavaScript. It allows users to convert an amount from one currency to another using live exchange rates fetched from an API.

## Features

- Convert amounts between different currencies
- Choose currencies from dropdown menus
- Display country flags alongside currency options
- Show the converted amount instantly after clicking the button
- Validate input and handle invalid values gracefully

## Technologies Used

- HTML5
- CSS3
- JavaScript
- exchangerate API for real-time exchange rates
- Flag API for currency flag display

## Project Structure

- index.html - Main structure of the web page
- style.css - Styling for the user interface
- script.js - Main JavaScript logic for the converter
- app.js - Additional JavaScript logic and event handling
- codes.js - Currency and country code data

## How to Run

1. Clone or download this project to your computer.
2. Open the project folder.
3. Launch index.html in your browser.

You can also use a live server extension in VS Code for a smoother experience.

## Usage

1. Enter the amount you want to convert.
2. Select the source currency and target currency.
3. Click the "Get Exchange rate" button.
4. The converted amount will be displayed on the screen.

## Notes

- An internet connection is required to fetch live exchange rates.
- The app uses default currencies of USD to PKR when first opened.
